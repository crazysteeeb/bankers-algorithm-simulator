# bankers-algorithm-simulator
C# implementation of the Banker's Algorithm for deadlock avoidance and safe-state detection in operating systems.

## Overview

This project simulates how operating systems allocate resources safely while avoiding deadlock conditions.

The program evaluates system states and determines whether resource allocation requests can be completed safely.

## Features

- Safe-state detection
- Need matrix calculation
- Resource allocation tracking
- Deadlock avoidance simulation
- File/input parsing
- Algorithmic resource management

## Technologies Used

- C#
- .NET
- Visual Studio Code
- Operating Systems concepts

## Concepts Demonstrated

- Deadlock avoidance
- Banker's Algorithm
- Resource allocation
- Matrix operations
- Systems programming concepts
- Algorithm implementation
